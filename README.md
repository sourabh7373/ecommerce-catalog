# E-Commerce Catalog

A simple **E-Commerce Product Catalog** built with **Next.js 13**, **React**, **Tailwind CSS**, and **JSON Server** as a mock API.  
Users can view products and add them to the cart. The cart count updates in real-time.

---

## **Features**

- Display a list of products fetched from a mock API (JSON Server).  
- Add products to the cart and see cart count update instantly.  
- Fully responsive design using **Tailwind CSS**.  
- Built using **Next.js 13** (App Directory + TypeScript).  
- Mock backend powered by **JSON Server** for quick testing.  

---

### **1. Install Dependencies**
